# Student Attendance System

A C-based program designed to track and manage student attendance records.

## Features
* Records student attendance status.
* Process tracking logic efficiently.

## How to Run
To compile and run the program locally, use your terminal:
```bash
gcc attendance.c -o attendance
./attendance
```
